MUIO ver.5.6.0, 20260601

  1. Func Indicator framework — users can now define custom indicators per case (e.g., production intensity by technology) that flow through to the result viewer.
  2. Func Dual variables (shadow prices for emission limits, energy balance, and user-defined constraints) now live in a dedicated Duals.json data file and persist through the Save flow.
  3. Tech Solver discovery extended with macOS Homebrew and Linux distro paths as a fourth fallback tier. The solver subprocess now only sets cwd to the solver folder when running a bundled binary, so system installs no longer run from /usr/bin.
  4. Code Data-driven schema: Variables.json, Duals.json, Indicators.json, and Parameters.json now carry setrelation and name per entry, matching MUIO 5.6. Legacy Config.VARIABLES_C/Config.DUALS Python dicts removed.
  5. Code Backend cleanup: removed roughly 1,500 lines of dormant backup methods from DataFileClass; OsemosysClass refactored to use the shared HelpersClass for parameter, variable, and indicator building.
